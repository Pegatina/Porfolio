Ejercicio de HTML

En este ejercicio, vamos a utilizar algunos elementos básicos de HTML a nivel de texto, enlaces y propiedades CSS. Todo el estilo debe definirse mediante CSS, no se deben utilizar atributos HTML ni elementos como <font> para ello.

Mediante notas como estas se explicará el estilo que se debe dar a cada parte del documento. Concretamente, estas notas irán en color rojo, con tamaño de fuente de 10 puntos, y con márgenes a izquierda y derecha de 2 centímetros.

Los títulos aparecerán centrados y rodeados por un borde azul.

Texto

En esta sección, el texto aparecerá justificado.

En HTML existen varios elementos para dar formato al texto. En realidad, utilizando hojas CSS ninguno de estos elementos es necesario, pues se pueden asignar todas las propiedades de estilo mediante propiedades CSS. Por ejemplo, podemos probar a escribir una palabra en negrita de 2 maneras; utilizando el elemento <b> (<b>palabra<b>) o utilizando una propiedad CSS (<span style="font-weight: bold">palabra</span>).

Ahora prueba tú a aplicar mediante propiedades CSS algunos estilos a la fuente texto, como:
Distintas fuentes: Arial, Tahoma, Comic Sans MS...
Cursiva, VERSALES, diferentes grados de peso para conseguir negrita. Y COMBINACIONES DE VARIAS DE ELLAS.
Hay otros efectos para el texto, como subrayado, superrayado, tachado, parpadeante (Internet Explorer no muestra este efecto). También se puede definir el espaciado entre las palabras y entre las letras. Otra posibilidad es modificar la alineación vertical del texto(también sirve para otros tipos de elementos).

Con otras propiedades se pueden definir la alineación de los párrafos (izquiera, centrada, derecha o justificada), sangría de primera línea, o el interlineado.
También existe la posibilidad de pasar un texto a Mayúsculas, a minúsculas, o poner en mayúsculas la primera letra de cada palabra.

El párrafo anterior tiene un interlineado de 1,5 líneas.
